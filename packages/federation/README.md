# @lumenchain/federation

Zero-dependency **agent interconnection layer** — connect many agents (and machines) into one federated brain.

- **A2A messaging** — agent-to-agent protocol.
- **Agent-card generation** — publish an agent's capabilities/identity.
- **Light-chain federation** — share one append-only verifiable chain across brains.
- **Cross-spine dispatch** — route work across federated cognitive spines.

## Install
```
npm i @lumenchain/federation
```

## Exports
`./federation` · `./dispatch` · `./agent-card` · `./a2a`

## Design
Many parallel autonomous brains on one shared neural substrate — no master/slave. A2A and agent-cards make capabilities discoverable across the federation.
